## üéµ CUSTOM SONGS - VALUE & DIFFERENTIATION

### Current State
- Built-in songs limited to library

### VIP Value
- **Monthly**: 100 custom songs (covers most use cases)
- **Annual**: 500 custom songs (semi-professional use)

### Implementation
1. User uploads MP3 or YouTube URL
2. Firebase Storage stores files
3. SongService converts to playable format

5. Cost: ~$0.10-0.50 per hosted party in storage

### Revenue Implications
- Unlimited uploads (fixed cost to user)

- Marginal profit from upsell feature
