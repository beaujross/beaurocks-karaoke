# Yes/No Interview (Game Spirit)

Use this as a running interview log.  
Answer each question with `Yes` or `No`.

## Round 1: Global Philosophy

1. Should every game mode prioritize social energy over strict fairness?
2. Should host workflow speed be prioritized over granular pre-queue control?
3. Should all "fast actions" have an undo path by default?
4. Should participation rewards generally outweigh pure skill rewards?
5. Should we avoid mechanics that publicly highlight low performers?

## Round 2: Competition Tone

6. Should competition feel "light rivalry" rather than "hard esports"?
7. Should winners be celebrated briefly, without long loser-focused states?
8. Should game pacing favor short rounds over long deep rounds?
9. Should ties usually resolve with shared reward instead of tiebreak pressure?
10. Should host have override power for all voting outcomes?

## Round 3: Safety + Moderation

11. Should photo-based modes require stricter moderation than text/voting modes?
12. Should risky user submissions default to hidden until approved?
13. Should we block or down-rank repeat spam actions automatically?
14. Should offensive content handling prioritize immediate removal over warning-first?
15. Should hosts be able to permanently suppress a user for one session?
